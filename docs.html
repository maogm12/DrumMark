Getting Started

Drum Notation Guide

A text-first way to write professional drum scores. The language is designed to be as fast as typing but as powerful as traditional notation software.

Document Metadata

Set up your score's global information and timing.

Every score begins with a header section. These fields define the technical foundation and metadata of your music:

  • title, subtitle, composer: Essential credits that appear at the top of the exported PDF and score.
  • tempo: Sets the BPM (Quarter note). If omitted, it defaults to 120.
  • time: Defines the meter (e.g., 4/4, 6/8, 7/8). This is required for rhythmic validation.
  • divisions: Defines the grid resolution for one measure. For example, setting it to 16 allows you to write 16th-note patterns using a 16-slot grid.
Example
title Funk Study
subtitle Groove A
composer G. Mao
tempo 100
time 4/4
divisions 8

HH | x - x x x x x - |

Musical Grouping

Control beat structures and visual beaming.

The grouping field tells the system how to beam notes and where the internal accents fall. Common meters have smart defaults:

  • 4/4 → 2+2 (two half-note beats)
  • 6/8 → 3+3 (two dotted-quarter beats)
  • Irregular meters: For 5/8, 7/8, etc., you must specify the grouping (e.g., 2+2+3).
Example
time 7/8
divisions 7
grouping 2+2+3

HH | x x x x x x x |

Instruments & Tracks

Map your ideas to specific percussion parts using a wide range of supported tracks.

Each instrument is represented by a track. Once a track is declared in any paragraph, it remains active throughout the score. If omitted in later sections, it is auto-filled with rests.

  • Cymbals: HH (Hi-Hat), RC (Ride), C (Crash). Use x/X.
  • Drums: SD (Snare), BD (Bass), T1/T2/T3 (Toms). Use d/D.
  • Foot & Utility: HF (Hi-Hat Foot) uses p/P for pedal hits. ST is for sticking (R/L).
  • Sugar Tracks: DR expands into SD and T1-T3. It supports accents (S, T1-T3).
Example
time 4/4
divisions 16

HH | x - x - x - x - | x - x - x - x - |
SD |  - - - - d - - - | - - - - d - - - |
BD | p - - - - - - - | p - - - - - - - |

Playing Techniques

Add detail to your hits with modifiers.

Refine notes with :modifier. Strict compatibility rules apply:

  • Snare (SD) & Toms (T1-T3): :rim, :cross, :flam.
  • Hi-Hat (HH): :open (shorthand o/O), :close.
  • Cymbals (C, RC): :choke, :bell (Ride only).
  • Foot (HF) & Bass (BD): :close (HF only).
  • Ghost Notes: Use the :ghost modifier to render noteheads with parentheses.
Example
time 4/4
divisions 8
grouping 1+1+1+1

SD | d:cross - d:rim - d:flam - d:ghost - |
RC | x:bell - x:choke - x:bell - x:choke - |
Score Result

Syntax & Formatting

Writing clean, maintainable notation code.

The notation language is designed to be human-readable and flexible:

  • Comments: Use # to add notes for yourself.
  • Whitespace: Spaces and tabs are ignored inside measures.
  • Barlines: You can place multiple measures on one line.
Example
# Cleanly aligned code
time 4/4
divisions 8

HH | x x x x  x x x x |
SD | - - d -  - - d - |
BD | p - - -  p - - - |

Rhythmic Groups

Adjust durations and create complex subdivisions using brackets.

Groups allow you to deviate from the base grid defined by divisions:

  • Compression (Tuplets): [2: d d d] creates a triplet.
  • Expansion (Longer Notes): [2: p] makes the note last for 2 slots.
Example
time 4/4
divisions 8
grouping 1+1+1+1

# Compression
SD | [d d] d [2:d d d d] [2: d d d] [2: d d d d d] |

# Expansion
SD | [2: d] d d [4: d] | [8: d] |
Score Result

Duration Suffixes

Fine-tune note lengths without brackets.

You can append suffixes directly to notes to adjust their length:

  • d.: A dotted note (1.5x length).
  • d/: A halved note (0.5x length).
Example
time 4/4
divisions 4
grouping 1+1+1+1

# Perfect combination
HH | x. x/ x. x/ | x. / x. / |
SD | - - d. d/ | d. / d d/ / |
Score Result

Structure & Flow

Manage bars, repeats, and sections.

Use |: and :| for repeats. :|x3 specifies total plays.

Example
time 4/4
divisions 8

HH |: x x x x x x x x :|x4
SD |  - - d - - - d -  |
BD |  p - - - p - - -  |

The Master Class

A complete demonstration using all available features.

This final example showcases the synergy of headers, complex groupings, shortcuts, and multiple paragraphs.

Example
title Fusion Grooves
subtitle Advanced Study
composer G. Mao
tempo 128
time 4/4
divisions 16
grouping 2+2

# Section A
HH |: x - x - o - x - | x:close - X - x - c - :|x2
