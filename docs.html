Getting Started

Drum Notation Guide

A text-first way to write professional drum scores. The language is designed to be as fast as typing but as powerful as traditional notation software.

Document Metadata

Global information and timing.

Set up credits, tempo, and time signature.

Example
title Funk Study\ntempo 100\ntime 4/4\ndivisions 8\n\nHH | x - x x x x x - |
Score Result

Musical Grouping

Control visual beaming.

Defines how notes are grouped and beamed.

Example
time 7/8\ndivisions 7\ngrouping 2+2+3\n\nHH | x x x x x x x |
Score Result

Instruments & Tracks

Mapping percussion parts.

Supports HH, RC, C, SD, BD, T1-T3, HF, ST.

Example
time 4/4\ndivisions 8\n\nHH | x - x x x x x x |\nSD | - - d - - - d - |\nBD | p - p - p - - - |
Score Result

Playing Techniques

Adding detail with modifiers.

Use :rim, :cross, :flam, :ghost.

Example
time 4/4\ndivisions 8\n\nSD | d:cross - d:rim - d:flam - d:ghost - |\nRC | x:bell - x:choke - x:bell - x:choke - |
Score Result

Syntax & Formatting

Clean notation code.

Flexible whitespace and comments.

Example
# Cleanly aligned\ntime 4/4\ndivisions 8\n\nHH | x x x x  x x x x |\nSD | - - d -  - - d - |
Score Result

Rhythmic Groups

Adjust durations with brackets.

Supports subdivisions and tuplets.

Example
time 4/4\ndivisions 8\n\nSD | [d d] d [2:d d d d] [2: d d d] |
Score Result

Duration Suffixes

Fine-tune note lengths.

Dotted (.) and halved (/) suffixes.

Example
time 4/4\ndivisions 4\n\nHH | x. x/ x. x/ | x. / x. / |
Score Result

Structure & Flow

Bars and repeats.

Repeat counts and sections.

Example
time 4/4\ndivisions 8\n\nHH |: x x x x x x x x :|x4\nSD |  - - d - - - d -  |
Score Result

The Master Class
