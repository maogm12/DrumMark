DrumMark Guide

A text-first way to write professional drum scores.

Overview

DrumMark is designed for musicians who want to write scores as fast as they type.

DrumMark lets you write drum scores using simple text. You write what you want to hear, and the system turns it into professional sheet music.

The format is designed to be easy to read — even without a computer, you can look at a DrumMark document and understand the groove. It is also reliable: the same input always produces the same output, so you know exactly what you're getting.

Headers & Metadata

Defining the technical foundation of your score.

Every score starts with a few basic facts that tell the system how to interpret your music. These go at the very top of your document, before any notes.

  • title / subtitle / composer: Your score's name and credits. These appear on the printed sheet.
  • tempo: Quarter-note BPM. Default is 120 if omitted.
  • time: The meter (e.g., 4/4, 3/4, 6/8). Required.
  • divisions: Total grid units per measure. Required. Controls rhythmic precision — use 16 for standard precision, 32 for complex syncopation.
  • grouping: Defines the beat structure within a measure. When omitted, DrumMark infers it from the time signature: 4/4 → 2+2, 6/8 → 3+3, 3/4 → 1+1+1. For irregular meters like 5/8, grouping must be specified explicitly.

Tracks & Instruments

Mapping notes to specific drum kit components.

Each line in your score represents one instrument. You write the instrument name followed by a vertical bar, then the notes for that instrument across the measure.

Tokens & Rhythms

The atomic building blocks of your notation.

Each symbol in a measure represents one hit or one rest. The same symbol can look different depending on which instrument it's for — that's why you don't need to specify "cymbal" or "drum" for every note.

  • d: A normal hit. Shows as a filled notehead on drums, or an X on cymbals.
  • -: A rest.
  • x: A cymbal hit, or cross-stick on snare (stick tip on rim, head silent).
  • s / b / r / c: Shortcuts for snare, bass drum, ride, and crash. s is equivalent to SD:d, b to BD:d.
  • o: Open hi-hat.
  • p: Pedal hi-hat (heel-down, no stick).
  • g: Ghost note — soft, parenthesized.
  • t1 - t4: Tom hits (high to low).

Duration modifiers: A trailing dot extends the note by half its value (dotted). A slash halves it (sixteenth note when the base is a quarter).

Example
title Hit Types
time 4/4
divisions 8
grouping 4

HH | x - x - x - x - |
SD | s - d - s - d - |
BD | b - b - b - b - |

Modifiers & Techniques

Adding detail and articulation to your hits.

Add extra details to your hits by putting a colon after the note, then the technique name. You can chain several together if you need to.

  • :accent: Accent — heavier attack, rendered with >.
  • :ghost: Ghost note — soft, parenthesized.
  • :flam: Flam — a 16th-note grace note preceding the main hit.
  • :drag: Drag — two 16th-note grace notes preceding the main hit.
  • :roll: Roll — tremolo, indicated with slashes on the stem.
  • :dead: Dead stroke — muted, small x notehead.
  • :open / :half-open: Hi-hat pedal state (open / half-open).
  • :bell: Ride bell — hit the raised center of the cymbal.
  • :choke: Choke — mute the cymbal immediately after striking.
  • :rim / :cross: Snare rimshot or cross-stick.
Example
title Articulation Techniques
tempo 96
time 4/4
divisions 16
grouping 2+2

HH | d:open - d:close - d:ghost - d:accent - |
SD | d:accent - d:rim - d:cross - d:flam - |
BD | d:roll - d:dead - d - d |

Rhythmic Groups

Tuplets and custom rhythmic subdivisions.

Sometimes you need a rhythm that doesn't fit the regular grid — like three notes in the space of two. Use square brackets to group notes together and stretch or compress them to fit your time span.

  • [note1 note2 note3]: Three notes played evenly within the space of one beat. This is the shorthand for a triplet.
  • [3: note1 note2 note3]: Same thing — three notes in three units of time. The 3: means "fit these into three slots."
  • [5: note1 note2]: Fit two notes into five slots — this stretches them beyond their normal length.
  • Nested rhythms: You can put groups inside groups for very complex patterns, like a triplet within a triplet.
Example
title Rhythmic Groups
time 4/4
divisions 8
grouping 4

HH | x x x x x x x x |
SD | - - [2: d d d] - - - - |
BD | b - - - b - - - |

Combined Hits

Playing multiple parts simultaneously.

When you want multiple instruments to play at the same moment, write them together with a plus sign. This is called a combined hit. Both notes start at exactly the same time and have the same duration.

For example, x+s means the hi-hat and snare play together. You can combine three or more instruments too: x+s+b plays hi-hat, snare, and bass drum all at once.

Example
title Simultaneous Hits
time 4/4
divisions 8
grouping 4

HH | x+s - x+b - x+s - x+b - |
SD | - - d+BD:d - - - - |

Sticking

Hand assignments for complex patterns.

When you need to show which hand plays which note, use the ST track. Write R for right hand and L for left hand. They line up with the notes above them, so the player knows exactly what to do.

ST annotations are especially useful for complex patterns, jazz charts, or any music where hand placement matters.

Example
title Hand Coordination
tempo 90
time 4/4
divisions 8
grouping 4

SD | d d d d d d d d |
ST | R L R L R L R L |

Repeats & Jumps

Structural navigation and alternative endings.

DrumMark supports all the standard ways to repeat music and jump around your score:

  • Repeat signs: |: starts a repeat, :| ends it. When the music reaches the end sign, it goes back to the start. Playback happens twice by default.
  • Alternative endings: After a repeat, you can have multiple endings. Use |1. for the first ending and |2. for the second. The player plays ending 1, then repeats, then plays ending 2.
  • Measure repeat: Write % all by itself in a measure to repeat the entire previous measure. Use %% to repeat two measures back.
  • Navigation markers: You can mark places with @segno (the sign), @coda (the coda), or @fine (the end). Then use jumps like @da-capo (go back to the beginning), @dal-segno (go back to the segno), or @to-coda (skip ahead to the coda).
Example
title Repeat Structure
time 4/4
divisions 8
grouping 4

HH |: x x x x |1. x x x x :|2. x x x x |
Score Result

Multi-Measure Rests

Standard notation for extended periods of silence.

When the whole band rests for several measures, use a multi-measure rest. Write a horizontal line with the number of measures in the middle: |--8--| means "rest for 8 measures." This renders as a single tall rest symbol spanning all eight bars.

Example
title Extended Rest
time 4/4
divisions 4
grouping 4

| --16-- |

Inline Repeats

Quickly Duplicate measures in place.

When you need the same measure several times in a row, you can write it once and repeat it. Put *4 at the end of a measure to repeat it four times total. This is handy for patterns that repeat exactly, like a repeating four-measure phrase.

Example
title Measure Duplication
time 4/4
divisions 8
grouping 4

HH | x x x x x x x x *4 |

Validation

Ensuring musical and logical consistency.

DrumMark checks your music for mistakes so you don't end up with confusing or unplayable sheet music. The system validates:

  • Measure length: All the notes and rests in a measure must add up to exactly the right amount. If you have too many or too few, the system will tell you.
  • Beat boundaries: Notes can't "cross" into the next beat area. This keeps your rhythm clear and readable.
  • Technique rules: Some techniques only work on certain instruments. For example, :open only makes sense on hi-hat — if you write it on snare, DrumMark will let you know.
