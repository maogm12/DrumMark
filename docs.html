Getting Started

Drum Notation Guide

A text-first way to write professional drum scores. The language is designed to be as fast as typing but as powerful as traditional notation software.

Document Metadata

Set up your score's global information and timing.

Every score begins with a header section. These fields define the technical foundation and metadata of your music:

  • title, subtitle, composer: Essential credits that appear at the top of the exported PDF and score.
  • tempo: Sets the BPM (Quarter note). If omitted, it defaults to 120.
  • time: Defines the meter (e.g., 4/4, 6/8, 7/8). Required for rhythmic validation.
  • divisions: Defines the grid resolution per measure (e.g., 16 for 16th-note resolution).
  • grouping: (Optional) Defines beat structures and beaming (e.g., 2+2, 3+3+2).
Example
title Funk Study
subtitle Groove A
composer G. Mao
tempo 100
time 4/4
divisions 16

HH | x - x - x - x - |

Musical Grouping

Control beat structures and visual beaming.

The grouping field tells the system how to beam notes and where the internal accents fall. Common meters have smart defaults:

  • 4/4 → 2+2 (two half-note beats)
  • 6/8 → 3+3 (two dotted-quarter beats)
  • Irregular meters: For 5/8, 7/8, etc., you must specify the grouping (e.g., 2+3 or 3+2+2).
Example
time 7/8
divisions 7
grouping 2+2+3

HH | x x x x x x x |

Instruments & Tracks

Map your ideas to specific percussion parts using a wide range of supported tracks.

Each instrument is represented by a track. Once a track is declared, it remains active throughout the score.

  • Cymbals: HH, RC, RC2, C, C2, SPL, CHN.
  • Drums: SD, BD, BD2, T1, T2, T3, T4.
  • Percussion: CB (Cowbell), WB (Wood Block), CL (Claps).
  • Utility: HF (Hi-Hat Foot), ST (Sticking: R/L).
Example
time 4/4
divisions 16

HH | x - x - x - x - |
SD | - - d - - - d - |
BD | b - - - - - - - |
CB | - - - - d - - - |
BD2 | - - - - - - b - |

Playing Techniques

Add detail to your hits with modifiers.

Refine notes with :modifier. Supported techniques:

  • Snare/Toms: :rim, :cross, :flam, :drag, :roll, :ghost, :dead.
  • Hi-Hat: :open (o), :half-open, :close, :ghost.
  • Cymbals: :choke, :bell (Ride only), :roll.
Example
time 4/4
divisions 16

HH | d - d:half-open - d:open - d:close - |
SD | d:flam - d:drag - d:roll - d:ghost - |
RC | d:bell - d:roll - d:bell - d:choke - |

Syntax & Formatting

Writing clean, maintainable notation code.

The notation language supports advanced layout features:

  • Multi-measure rest: Use | --N-- | (e.g., | --8-- |).
  • Inline Repeat: Use *N to repeat a measure (e.g., | d d d d *4 |).
  • Comments: Use # for personal notes.
Example
time 4/4
divisions 16

HH | x - x - x - x - *2 | # Repeat measure 2 times
HH | --4-- | # 4-measure rest

Rhythmic Groups

Adjust durations and create complex subdivisions using brackets.

Groups allow you to deviate from the base grid:

  • Tuplets: [2: d d d] creates a triplet over 2 slots.
  • Compression/Expansion: [1: d d d d] fits 4 notes into 1 slot.
Example
time 4/4
divisions 8

SD | [d d] d [2: d d d] [2: d d d d] |
BD | [2: d] d d [4: d] |
Score Result

Duration Suffixes

Fine-tune note lengths without brackets.

Append suffixes directly to notes to adjust their length:

  • d.: Dotted note (1.5x length).
  • d/: Halved note (0.5x length). Multiple / or . accumulate.
Example
time 4/4
divisions 4

HH | x. x/ x. x/ |
SD | - - d. d/ |

Structure & Flow

Manage bars, repeats, and sections.

Control the score's navigation and measure repeats:

  • Barlines: |: and :| for repeats. |1., |2. for voltas.
  • Measure Repeat: % repeats 1 measure, %% repeats 2.
  • Navigation: @segno, @coda, @dal-segno, @to-coda, @fine.
Example
time 4/4
divisions 16

HH |: @segno x - x - x - x - | % :|
SD | - - d - - - d - | % :|
BD | b - - - - - - - | % :| @to-coda

The Master Class

A complete demonstration using all available features.

This final example showcases the synergy of headers, complex tracks, modifiers, and navigation.

Example
title Fusion Grooves
subtitle Advanced Study
composer G. Mao
tempo 128
time 4/4
divisions 16
grouping 2+2

# Intro
HH |: @segno x - x - o - x - | x:close - X - x - c - :|
SD | - - d:cross - d - | D:rim - [2: d d:flam] - - - |
BD | b - - - b - - - | b - b - - - b - | @to-coda

# Bridge
HH | x - x - x - x - | *2
SD | d:roll - - - - - - - | --2-- |

# Outro
@coda
C | c:accent - - - - - - - | @fine
