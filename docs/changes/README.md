# Changes

New change records live here.

Active changes:

```text
docs/changes/active/<change-id>/
  plan.md
  history.md
```

Completed changes are archived under:

```text
docs/changes/archive/<year>/<change-id>/
```

Do not create loose `plan.md` or `history.md` files.

Do not create new proposal/tasks files under `docs/proposals/` for new work.
