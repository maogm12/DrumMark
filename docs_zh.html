快速入门

鼓谱记谱指南

为现代鼓手设计的“文本优先”记谱方式。

乐谱元数据

设置乐谱的全局信息和节奏基础。

包含标题、作者、速度（BPM）、拍号和细分数。

示例
title 灵魂乐研究\ntempo 100\ntime 4/4\ndivisions 8\n\nHH | x - x x x x x - |
生成结果

乐句分组

控制符杠连接逻辑。

定义内部重音和视觉分组。

示例
time 7/8\ndivisions 7\ngrouping 2+2+3\n\nHH | x x x x x x x |
生成结果

乐器与轨道

将创意映射到声部。

支持 HH, RC, C, SD, BD, T1-T3, HF, ST 等标准轨道。

示例
time 4/4\ndivisions 8\n\nHH | x - x x x x x x |\nSD | - - d - - - d - |\nBD | p - p - p - - - |
生成结果

演奏技巧

通过修饰符丰富细节。

支持 :rim, :cross, :flam, :ghost 等。

示例
time 4/4\ndivisions 8\n\nSD | d:cross - d:rim - d:flam - d:ghost - |\nRC | x:bell - x:choke - x:bell - x:choke - |
生成结果

语法与格式

保持代码整洁。

支持 # 注释和自由空格对齐。

示例
# 整齐对齐的代码\ntime 4/4\ndivisions 8\n\nHH | x x x x  x x x x |\nSD | - - d -  - - d - |
生成结果

连音符与分组

利用方括号调整时长。

支持细分（压缩）和长音符（扩展）。

示例
time 4/4\ndivisions 8\n\nSD | [d d] d [2:d d d d] [2: d d d] |
生成结果

时长后缀

快速调整音符长度。

使用附点 (.) 和减半 (/) 后缀。

示例
time 4/4\ndivisions 4\n\nHH | x. x/ x. x/ | x. / x. / |
生成结果

结构与流程

管理重复与段落。

支持重复次数标记和多乐段布局。

示例
time 4/4\ndivisions 8\n\nHH |: x x x x x x x x :|x4\nSD |  - - d - - - d -  |
生成结果

综合特性演示。
