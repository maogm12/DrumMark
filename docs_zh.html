鼓谱记谱指南

为现代鼓手设计的文本优先记谱系统。

乐谱元数据

设置乐谱的全局信息和节奏基础。

每一份乐谱都从小节头（Header）开始。这些字段定义了音乐的技术基础和元数据：

  • title, subtitle, composer: 必填的乐谱信息。
  • tempo: 设置 BPM（四分音符为准）。
  • time: 定义拍号（如 4/4）。
  • divisions: 定义每一小节的网格细分数。
示例
title 灵魂乐研究
subtitle 律动 A
composer G. Mao
tempo 100
time 4/4
divisions 8

HH | x - x x x x x - |

乐句分组

控制音符的符杠连接和视觉重音。

grouping 字段是提高乐谱可读性的最强工具。4/4 默认为 2+2。

示例
time 7/8
divisions 7
grouping 2+2+3

HH | x x x x x x x |
生成结果

乐器与轨道

将你的创意映射到具体的打击乐声部。

标准轨道：HH, RC, C, SD, BD, T1-T3, HF, ST。

示例
time 4/4
divisions 8

C  | x - - - - - - - |
HH | - x x x x x x x |
SD | - - d - - - d - |
BD | p - p - p - - - |
生成结果

演奏技巧

通过修饰符丰富打击感。

支持：:rim, :cross, :flam, :ghost 等。

示例
time 4/4
divisions 8
grouping 1+1+1+1

SD | d:cross - d:rim - d:flam - d:ghost - |
RC | x:bell - x:choke - x:bell - x:choke - |
生成结果

语法与格式细节

编写整洁、可维护的记谱代码。

支持 # 注释 和空格对齐。

示例
# 整齐对齐的代码

divisions 8

HH | x x x x  x x x x |

连音符与分组

利用方括号调整时长或创建复杂的细分。

使用 [跨度: 音符列表] 进行智能展开。

示例
time 4/4
divisions 8
grouping 1+1+1+1

# 压缩
SD | [d d] d [2:d d d d] [2: d d d] |

# 扩展
SD | [2: d] d d [4: d] |
生成结果

结构与流程

管理小节、重复和乐段。

支持 |:, :|x4 标记和多乐段布局。

示例
time 4/4
divisions 8

HH |: x x x x x x x x :|x4
SD |  - - d - - - d -  |

大师级全特性示例

完整演示。

涵盖装饰音、镲帽、粘滞标记等高级技巧。

示例
title Fusion Grooves
subtitle 高级进阶练习
composer G. Mao
tempo 128
time 4/4
divisions 16
grouping 2+2

# 乐段 A: 主律动
HH |: x - x - o - x - | x:close - X - x - c - :|x2
SD |  - - d:cross - d - | D:rim - [2: d d:flam] - - -  |
BD |  p - - - p - - - | p - p - - - p -        |
HF |  - - - - p - - - | - - - - p:close - -    |

# 乐段 B: 复杂细分桥接
RC |  x:bell - x:bell - x:bell - x:bell - | [4: x:choke] |
DR |  s - - - [3: s s s] - - - | S - t1 t2 t3 - - -   |
BD |  p - - - p - - -     | p - - - p - - -      |
ST |  R - - - R L R - - - | R - R L R - - -      |

# 结尾: Finale
C  |  X:choke - - - - - - - | - - - - X - - - |
BD |  [16: p] |
生成结果
