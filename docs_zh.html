鼓谱记谱指南

为现代鼓手设计的“文本优先”记谱方式。该语言旨在像打字一样飞快，同时拥有传统打谱软件的强大功能。

乐谱元数据

设置乐谱的全局信息和节奏基础。

每一份乐谱都从小节头（Header）开始。这些字段定义了音乐的技术基础和元数据：

  • title, subtitle, composer: 必填的乐谱信息，将显示在导出的 PDF 和乐谱顶部。
  • tempo: 设置 BPM（四分音符为准）。如果省略，默认为 120。
  • time: 定义拍号（如 4/4, 6/8, 7/8）。这是节奏校验的依据。
  • divisions: 定义每一小节的网格细分数。例如，设置为 16 可以让你在 16 格的网格中编写 16 分音符。
示例
title 灵魂乐研究
subtitle 律动 A
composer G. Mao
tempo 100
time 4/4
divisions 8

HH | x - x x x x x - |
生成结果

乐句分组

控制音符的符杠连接和视觉重音。

grouping 字段是提高乐谱可读性的最强工具。它告诉系统如何连接符杠。常用拍号拥有智能默认值：

  • 4/4 → 2+2
  • 6/8 → 3+3
  • 非正规拍号: 对于 5/8, 7/8 等，你必须指定分组（如 2+2+3）。
示例
time 7/8
divisions 7
grouping 2+2+3

HH | x x x x x x x |

乐器与轨道

将你的创意映射到具体的打击乐声部。

每种乐器由一个轨道（Track）表示。一旦轨道在某个段落出现，它将在整个乐谱中持续存在。

  • 镲片 (Cymbals): HH (踩镲), RC (叮叮镲), C (吊镲)。
  • 鼓组 (Drums): SD (军鼓), BD (底鼓), T1/T2/T3 (通通鼓)。
  • 底端与功能: HF (脚踏踩镲), ST (粘滞标注 R/L)。
示例
time 4/4
divisions 8

C  | x - - - - - - - |
HH | - x x x x x x x |
SD | - - d - - - d - |
BD | p - p - p - - - |
生成结果

演奏技巧

通过修饰符丰富打击感。

使用 :修饰符 语法。支持：:rim (边击), :cross (横跨), :flam (装饰音), :ghost (虚音)。

示例
time 4/4
divisions 8
grouping 1+1+1+1

SD | d:cross - d:rim - d:flam - d:ghost - |
RC | x:bell - x:choke - x:bell - x:choke - |
生成结果

语法与格式细节

编写整洁的代码。

该记谱语言旨在提供极高的可读性：支持 # 注释，忽略空格对齐。

示例
# 整齐对齐的代码
time 4/4
divisions 8

HH | x x x x  x x x x | # 主歌
SD | - - d -  - - d - | # 律动
生成结果

连音符与分组

利用方括号调整时长。

利用 [跨度: 音符] 进行压缩（连音符）或扩展（长音符）。

示例
time 4/4
divisions 8

# 压缩：32分音符、三连音
SD | [d d] d [2:d d d d] [2: d d d] |

# 扩展：四分音符
SD | [2: d] d d [4: d] |
生成结果

时长后缀

快速调整音符长度。

使用附点 (.) 增加 50% 时长，使用斜杠 (/) 减少一半时长。

示例
time 4/4
divisions 4
grouping 1+1+1+1

HH | x. x/ x. x/ | x. / x. / |
SD | - - d. d/ | d. / d d/ / |
生成结果

结构与流程

管理小节、重复和乐段。

使用 |: 和 :|。:|x4 标记会自动展开乐谱。

示例
time 4/4
divisions 8

HH |: x x x x x x x x :|x4
SD |  - - d - - - d -  |
BD |  p - - - p - - -  |
生成结果
